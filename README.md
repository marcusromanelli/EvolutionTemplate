# EvolutionTemplate

Evolution Template is a template project to aid the development of "Evolution" game types.
It can be easily setup by finding the "Assets/EvolutionTemplate/Settings/Game Library" file.

The game features:
* 4 evolvable elements (Due to the lack of sprites, I am using 4 types of cats)
* Coin/Cash accountability
* Encyclopedia with Icon, Name, Description and CoinsPerSecond ratio
* General Statistics
* Watch video for Ads
* Purchasable Skin support


The project was made in Unity, using C#
